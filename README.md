## RUN DI TERMUX/UBUNTU/SSH USER

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
apt install imagemagick -y
git clone https://github.com/KyuraSaja/doyanie-md
cd doyanie-md
npm install
npm start                                                                                                                                             
```
